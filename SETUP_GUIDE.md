# Trading Report Setup Guide

## Telegram (Recommended - Free, SMS-like)

### Steps:
1. Open Telegram, search for **@BotFather**
2. Send `/newbot` to create a bot
3. Copy the **bot token** (e.g., `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`)
4. Start a chat with your new bot
5. Get your **chat_id**: search for **@userinfobot**, send `/start`
6. Copy your chat_id (a number like `123456789`)

### Edit report_config.json:
```json
{
    "telegram": {
        "enabled": true,
        "bot_token": "YOUR_BOT_TOKEN_HERE",
        "chat_id": "YOUR_CHAT_ID_HERE"
    }
}
```

## Gmail API (No Password - OAuth2)

### Steps:
1. Go to: https://console.cloud.google.com/
2. Create a new project
3. Search for "Gmail API" and enable it
4. Go to "Credentials" → "Create Credentials" → "OAuth client ID"
5. Download the JSON file
6. Rename it to `gmail_oauth.json`
7. Place it in the same folder as `trading_report.py`

### Edit report_config.json:
```json
{
    "email": {
        "enabled": true,
        "receiver_email": "lulgabc@gmail.com",
        "use_gmail_api": true,
        "oauth_file": "gmail_oauth.json"
    }
}
```

## Free SMS Alternatives

### Option 1: Telegram (Best)
- Works like SMS on your phone
- Free, instant, reliable

### Option 2: Email to SMS (Free)
Most carriers offer free email-to-SMS:

| Carrier | Email Format |
|---------|-------------|
| AT&T | `1234567890@txt.att.net` |
| Verizon | `1234567890@vtext.com` |
| T-Mobile | `1234567890@tmomail.net` |
| China Mobile | `1234567890@139.com` |
| China Unicom | `1234567890@wo.com.cn` |

### Example:
```json
{
    "email": {
        "enabled": true,
        "receiver_email": "your_phone_number@vtext.com"
    }
}
```
